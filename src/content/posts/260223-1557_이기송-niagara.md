## 💬 자주 묻는 질문 (FAQ)

**Q. Sprite 폭발 FX 제작 시 Panning 애니메이션이 부자연스럽게 느껴집니다.**

게임제작 전문가 이기송: Panning 애니메이션의 부자연스러움은 주로 Lifetime과 Panning 그래프의 Timing 설정에서 오는 경우가 많습니다. Lifetime을 0.6초로 설정하고, Panning의 Handle Mode를 'Auto'로 변경하여 그래프를 조정해 보세요. 특히 Distortion 값이 처음부터 높게 설정되어 있으면 Panning에 방해가 될 수 있으니, 초반에는 0.1 정도로 낮게 시작했다가 점차 왜곡되는 느낌으로 조절하는 것이 좋습니다.

**Q. Sub Emitter의 Size와 Lifetime이 Core Emitter와 조화롭게 어우러지지 않는 것 같습니다.**

게임제작 전문가 이기송: Sub Emitter의 Size와 Lifetime은 Core Emitter와의 조화를 고려하여 설정해야 합니다. Core Emitter의 Lifetime이 0.6초라면, Sub Emitter는 이보다 조금 더 오래 살되(예: 0.7초), Size 변화 그래프를 활용하여 Core Emitter의 Panning과 비슷한 타이밍에 퍼지거나 사라지도록 조정하는 것이 좋습니다. 또한, Sort Order Hint 값을 조절하여 Emitter 간의 렌더링 우선순위를 명확히 해주면 더욱 자연스러운 블렌딩 효과를 얻을 수 있습니다.

**Q. Lens Flare와 같은 보조 이펙트 요소의 Color 값을 어떻게 조절해야 할까요?**

게임제작 전문가 이기송: Lens Flare와 같은 보조 이펙트 요소의 Color 조절은 Multiply Vector by Float 노드를 활용하는 것이 효과적입니다. Float from Curve 노드를 사용하여 Time에 따른 Color 값 변화를 세밀하게 조절할 수 있으며, 채널별(R, G, B)로 다른 Curve를 적용하여 원하는 색감과 채도를 표현할 수 있습니다. 예를 들어, 흰색에서 시작하여 점차 특정 색상으로 변하는 효과를 연출할 수 있습니다.
