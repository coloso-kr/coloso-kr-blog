## 🛠️ 실무 퀄리티를 결정짓는 핵심 스킬 & 디테일

Unity에서 iOS 빌드를 할 때, 생각보다 많은 디테일한 설정들이 필요합니다. 특히 **`Project Settings > Player > Other Settings`**에서 **`Override Default Bundle Identifier`**를 체크하고 Bundle ID를 `com.nadocoding.pandarun`처럼 정확히 설정하는 것은 필수입니다. 이 Bundle ID가 Apple Developer Program에서 등록한 ID와 일치해야 앱 스토어 심사 과정에서 문제가 발생하지 않아요. 또한, 게임의 **`Default Orientation`**을 **`Landscape`**로 설정하여 가로 모드로만 게임이 플레이되도록 제한하는 것이 일반적입니다.

**만약 게임 화면에서 좌우 레터박스가 생긴다면?**
- **해결책**: Unity의 **`Square`** 스프라이트를 새로 생성하여 **`LeftBlack`**, **`RightBlack`** 등의 이름으로 명명하고, **Sorting Layer**를 **`Character`**로, **Order Layer**를 **`5`** 정도로 설정한 뒤, **Sprite Renderer**의 **`Color`**를 **검은색**으로 바꿔주세요. 이후 **Ground**의 **Z 오더(-1)보다 앞서는 Z 오더(-2)**로 위치를 조정하면 됩니다.

**UI가 화면 밖으로 나간다면?**
- **해결책**: **Canvas**의 **`Canvas Scaler`** 설정에서 **`Screen Match Mode`**를 **`Match Width Or Height`**로 변경하고 **`Width`**와 **`Height`** 값을 조절하세요. 특히 **ScorePanel**과 같이 화면 좌측 상단에 고정되는 UI는 **Anchor Presets**를 **중앙**으로 설정한 후 **Alt 키**를 누른 상태로 **Scene 뷰**에서 원하는 위치로 직접 옮겨주는 것이 좋습니다.

## 💬 자주 묻는 질문 (FAQ)
**Q. Apple Developer Program 등록 비용은 얼마인가요?**

Apple Developer Program 등록 비용은 현재 기준으로 매년 129,000원입니다. 이는 1년마다 갱신해야 하며, 추후 출시 계획이 확실해졌을 때 등록하는 것을 권장합니다.

**Q. Unity 프로젝트의 Bundle ID는 어떻게 설정해야 하나요?**

Unity의 `File > Project Settings > Player > Other Settings`에서 **Company Name**과 **Product Name**을 설정한 후, `Override Default Bundle Identifier`를 체크하고 `com.회사명.게임이름` 형식에 맞춰 입력해야 합니다. 이 값은 Apple Developer Program에서 생성한 App ID와 일치해야 합니다.

**Q. iOS 빌드 후 Xcode에서 'Provisioning Profile' 오류가 발생하면 어떻게 해야 하나요?**

Xcode에서 `Signing & Capabilities` 설정으로 이동하여 `Automatically manage signing`을 체크하고 **Team**을 올바르게 선택해야 합니다. Provisioning Profile은 Apple Developer 사이트에서 앱 ID 생성 시 함께 만들어지며, Unity 빌드 설정과 Xcode 설정이 일관되게 유지되는 것이 중요합니다.

**Q. 게임 출시 전 UI가 다양한 해상도에서 잘 보이도록 하려면 어떻게 해야 하나요?**

Unity의 **Canvas Scaler** 컴포넌트에서 **`Screen Match Mode`**를 **`Match Width Or Height`**로 설정하고, **`Width`**와 **`Height`** 값을 조절하여 최적화하세요. 또한, UI 요소의 **Anchor Presets**를 **중앙**으로 고정하고 **Alt 키**를 누른 상태로 Scene 뷰에서 직접 위치를 조정하는 것이 좋습니다.
