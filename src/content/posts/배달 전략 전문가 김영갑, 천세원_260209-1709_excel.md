# 시트 작업, 엑셀 기초부터 실전 활용까지 완벽 마스터

## 🧐 엑셀 시트 작업, 뭔가 비효율적이라고 느낀 적 없으신가요?
![강의 이미지 1](https://storage.googleapis.com/static.fastcampus.co.kr/prod/uploads/202110/174453-465/commerce-delivery-sellingpoint-01.png)

분명 똑같이 함수를 사용하는데 왜 내 결과물은 다를까요? 혹시 함수 이름만 보고 대충 사용하고 있진 않으신가요? 엑셀 시트 작업은 기본기만큼이나 '올바른 이해'가 중요합니다. 원리를 정확히 알면 단순 반복 작업에서 벗어나 훨씬 효율적으로 원하는 데이터를 뽑아낼 수 있습니다.

## 💡 배달 전략 전문가 김영갑, 천세원의 실무 가이드: 핵심 원리

### 1. VLOOKUP vs INDEX-MATCH: 상황별 최적 함수 활용법
단순히 VLOOKUP 하나만 고집하는 건 시간 낭비일 수 있습니다. 검색할 데이터 범위가 넓거나, 참조할 열의 위치가 자주 바뀐다면 INDEX-MATCH 조합이 훨씬 유용합니다. 특히 `INDEX` 함수의 반환 열을 `MATCH` 함수로 동적으로 지정하면, 나중에 열을 추가하거나 삭제해도 수식을 수정할 필요가 없습니다. 이 두 함수의 장단점을 명확히 구분해서 사용하면 검색 속도와 유연성을 동시에 잡을 수 있습니다.

### 2. 데이터 유효성 검사: 입력 오류 99% 방지 팁
데이터를 입력할 때 가장 많이 발생하는 실수는 오탈자나 잘못된 형식입니다. '데이터 유효성 검사' 기능을 활용하면 이런 실수를 원천적으로 막을 수 있습니다. 예를 들어, 특정 셀에는 숫자만 입력되도록 설정하거나, 미리 지정된 목록에서만 선택하도록 제한하는 식이죠. 이렇게 설정해두면 작업 후 데이터를 검토하는 시간을 대폭 줄일 수 있습니다.

### 3. 조건부 서식: 시각적으로 핵심 데이터만 쏙쏙
방대한 데이터 속에서 특정 조건을 만족하는 셀을 일일이 찾으려면 눈이 아픕니다. 조건부 서식을 활용하면 이런 수고를 덜 수 있습니다. 예를 들어, 특정 값 이상인 셀은 자동으로 빨간색으로, 미만인 셀은 파란색으로 표시되도록 설정하면 한눈에 데이터의 추세를 파악할 수 있습니다. 단순히 색을 칠하는 것을 넘어, '데이터 막대'나 '아이콘 집합'을 활용하면 훨씬 직관적인 시각화가 가능합니다.

## 🛠️ 실무 필수 스킬 활용할 때 꼭 체크할 것들
![강의 이미지 2](https://storage.googleapis.com/static.fastcampus.co.kr/prod/uploads/202110/174456-465/commerce-delivery-sellingpoint-02.png)

VLOOKUP이나 INDEX-MATCH 함수를 사용할 때, 참조 범위 설정은 어떻게 하고 계신가요? 절대 참조(`$`)를 제대로 사용하지 않으면 수식을 복사했을 때 의도치 않은 결과가 나올 수 있습니다. 특히 시트 전체를 참조할 때는 `$A:$Z` 와 같이 열 전체를 참조하는 것이 편리할 때가 많습니다. 또한, 함수 입력 시 엑셀이 자동으로 추천하는 구문 완성 기능을 적극 활용하면 오타를 줄이고 빠르게 함수를 작성할 수 있습니다.

## 💬 자주 묻는 질문 (FAQ)

**Q. VLOOKUP 함수로 값을 찾는데 '#N/A' 오류가 떠요. 왜 그런가요?**
A. '#N/A' 오류는 찾으려는 값이 참조 범위에 존재하지 않을 때 발생합니다. 참조 범위에 동일한 값이 있는지, 혹시 공백이나 보이지 않는 문자가 포함되어 있지는 않은지 확인해보세요. `TRIM` 함수로 앞뒤 공백을 제거하는 것도 좋은 방법입니다.

**Q. INDEX-MATCH 함수를 쓸 때, MATCH 함수의 '검색 종류' 인수는 꼭 0으로 해야 하나요?**
A. '검색 종류'에 0을 입력하는 것은 정확히 일치하는 값을 찾겠다는 의미입니다. 대부분의 실무에서는 정확한 일치가 필요하므로 0을 사용합니다. 하지만 오름차순으로 정렬된 데이터에서 근사값을 찾고 싶다면 1을, 내림차순이라면 -1을 사용할 수도 있습니다.

**Q. 조건부 서식에서 여러 조건을 동시에 적용할 수 있나요?**
A. 네, 가능합니다. '새 규칙'에서 '수식을 사용하여 서식을 지정할 셀 결정'을 선택하고, `AND` 또는 `OR` 함수를 사용하여 여러 조건을 조합할 수 있습니다. 예를 들어, "A열 값이 100 이상이고, B열 값이 '완료'인 경우"처럼 복잡한 조건도 설정 가능합니다.

**Q. 데이터 유효성 검사에서 목록으로 지정한 항목 외에 다른 값을 입력하면 어떻게 되나요?**
A. 설정한 '오류 메시지' 창이 나타나며, 해당 셀에 입력을 허용하지 않습니다. 사용자가 어떤 값을 입력해야 하는지 명확히 안내하여 데이터의 일관성을 유지할 수 있습니다.
